
# Sudoku_By_EasyX

基于C语言和EasyX图形库的数独游戏。

## 功能特点

* 随机生成数独题目，每次进入游戏自动生成新的题目。
* 四种难度级别：简单、一般、困难、恐怖，影响空白格子的数量。
* 友好的图形界面：主菜单、游戏界面、设置界面、帮助界面。
* 支持鼠标交互：点击格子输入数字（1-9），点击提交答案检查正确性。
* 错误反馈：错误格子会变为红色，点击后恢复。
* 胜利画面：完成所有空格并正确后显示胜利界面。

## 开发环境

* Visual Studio 2022
* EasyX图形库（需自行安装）

## 如何运行

1. 安装Visual Studio 2022（或其它版本，但项目文件可能需要调整）。
2. 下载并安装EasyX库：[http://www.easyx.cn/](http://www.easyx.cn/) ，根据VS版本选择对应安装包。
3. 克隆或下载本项目源码。
4. 用VS打开项目文件（或创建空项目，添加源文件 `实验五.cpp`，并将Resource文件夹中的图片资源放到与可执行文件同目录下的 `Resource`文件夹中）。
5. 编译运行（注意字符集设置为多字节字符集，因为代码中使用 `outtextxy`输出中文，需要设置为使用多字节字符集）。

## 操作说明

* **主界面** ：点击“开始”进入游戏，点击“设置”调整难度，点击“帮助”查看帮助信息。
* **游戏界面** ：
* 点击空白格子（淡蓝色）后，按键盘数字键1-9填入数字。
  * 已填写的格子显示为浅绿色。
  * 点击“难度标签”（上方中间）可以快速回到设置界面。
  * 点击“提交答案”检查当前填写是否正确。
  * 如果答案错误，错误的格子会变为红色，点击任意位置后红色消失，可继续修改。
  * 如果答案正确，显示胜利画面，点击返回游戏界面。
  * 点击右下角的“返回”按钮返回主界面。
* **设置界面** ：点击选择难度，点击右下角“返回”返回上一页（主界面或游戏界面）。
* **帮助界面** ：显示帮助信息，点击右下角“返回”返回主界面。

## 注意事项

* 游戏依赖EasyX图形库，请确保正确安装。
* 图片资源文件需放在 `Resource`文件夹中，包括 `Game.png`, `MainPage.png`, `Sudoku_help.png`, `Victory.jpg`。如果缺失，程序可能无法正常显示背景。
* 由于使用了 `_kbhit()`和 `_getch()`，控制台输入与图形窗口并存，确保在游戏界面中点击格子后按数字键有效。
* 代码中使用 `pow(msg.x - 700, 2) + pow(msg.y - 800, 2) < 150 * 150`来判断是否点击了返回按钮，返回按钮位于窗口右下角（坐标700,800），注意窗口大小为700x800，所以右下角正好是边缘。

## 代码结构

* `main()`: 程序入口，页面循环。
* `LoadAllResource()`: 加载图片资源。
* `MainPage()`, `GamePage()`, `SetPage()`, `HelpPage()`: 各界面绘制与交互。
* `ReSetQuestion()`: 根据难度重新生成数独题目。
* `Check()`: 检查当前答案是否符合数独规则。
* `InDigit()`: 在9x9格子中绘制数字。
* `MyRand()`: 生成随机数。
